Educational Games

An educational games plugin for Moodle that aims to help in the learning process by motivating students with gamification aspects. Alongside with the plugin, a multiplayer trivia game that uses the plugin capabilities is provided as an example of usage. The game uses the plugin to get questions available from Moodle courses and creates a quiz game with multiplayer features that enables the competition between courses students. Download the project's .pdf (portuguese only) for more details. See media for pictures of the plugin and the game.

Technology: PHP, Javascript, HTML, CSS, JQuery, MySQL, phpMyAdmin, XAMPP, Moodle
